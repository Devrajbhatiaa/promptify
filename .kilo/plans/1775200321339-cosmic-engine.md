## Solution - Image Analysis Feature

### What This Does
- Users can upload images
- AI analyzes the image and provides description/caption
- Useful for understanding images, generating prompts from images, etc.

### Technical Approach
- Use Groq's vision-capable model: `llama-3.2-11b-vision-preview` or `llama-3.2-90b-vision-preview`
- Send image as base64 to the API
- Return AI description to user
